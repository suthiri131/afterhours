# After Hours

After Hours is a movie watchlist and review web application where users can keep track of movies they want to watch, mark them as watched, and leave ratings and reviews. This project is built as part of a group assignment and focuses on implementing core CRUD functionalities using Node.js, Express, and MongoDB.

## Core Functions

- Create a movie entry  
- Update a movie entry (mark as watched, add rating and review)  
- Delete a movie entry  
- View and manage saved movies  

## Features

- User authentication (register, login, logout)
- Personal movie watchlist
- Watched / not watched status
- Ratings and reviews
- Search and filter movies

## Tech Stack

- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- EJS  


## Getting Started
1. Clone the repository  
2. Install dependencies - npm i
3. Create a `config.env` file in the root directory and add: DB = your_mongodb_connection_string
4. Run the server - nodemon server.js
